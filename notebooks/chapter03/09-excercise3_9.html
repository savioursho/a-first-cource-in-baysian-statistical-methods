3.9 Galenshore分布#

import pandas as pd
import numpy as np
from scipy import special
import seaborn as sns
import warnings

sns.set_theme()
warnings.simplefilter("ignore")

\(\text{Galenshore}(a, \theta)\)

(1)#\[ p(y) = \frac{2}{\Gamma(a)} \theta ^{2a} y^{2a-1} e^{-\theta^2 y^2} \]

a#

Galenshore分布の密度関数 \(p(y)\) は指数型分布族の形 \(\kappa(n_0, t_0) c(\phi)^{n_0} e^{n_0 t_0 \phi}\) で書くことができる。

\[\begin{split} \begin{aligned} h(y) &=\frac{2 y^{2a - 1}}{\Gamma{a}} \\ c(\phi) &= \theta^{2a} \\ t(y) &= y^2 \\ \phi &= - \theta^2 \end{aligned} \end{split}\]

本文p.58から、この事前分布は

\[\begin{split} \begin{aligned} p(\theta | n_0, t_0) &= \kappa(n_0, t_0) \theta^{2a n_0} e^{- n_0 t_0 \theta^2} \\ &= \frac{2}{\Gamma(a n_0 + 1/2)} (n_0 t_0)^{a n_0 + 1/2} \theta^{2a n_0} e^{- n_0 t_0 \theta^2} \end{aligned} \end{split}\]

となることがわかり、これはGalenshore分布 \(\text{Galenshore} (a n_0 + 1/2, \sqrt{n_0 t_0})\)である。

いくつかのパラメータでGalenshore分布をプロットしてみる

# Galenshore分布の確率密度関数


@np.vectorize
def pdf_galenshore(y, a, theta):
    return (
        (2 / special.gamma(a))
        * (theta ** (2 * a))
        * (y ** (2 * a - 1))
        * np.exp(-(theta**2) * (y**2))
    )

params = [
    # (a, theta)
    (1, 1),
    (1, 2),
    (2, 1),
    (2, 2),
]

df_3_9 = pd.DataFrame()
df_3_9["theta"] = np.linspace(0, 4, 100)
for a, theta in params:
    df_3_9[f"a = {a}, theta = {theta}"] = pdf_galenshore(df_3_9["theta"], a, theta)

b#

本文p.53から、事後分布は \(\text{Galenshore}(a(n_0 + n) + 1/2, \sqrt{n_0 t_0 + n \bar{t}})\) であり、確率密度関数は

\[\begin{split} \begin{aligned} p(\theta | y_1, \cdots, y_n) &= p(\theta | n_0 + n, (n_0 t_0 + n \bar{t})/(n_0 + n)) \\ &= \frac{2}{\Gamma(a (n_0 + n) + 1/2)} (n_0 t_0 + n \bar{t})^{a(n_0+n) + 1/2} \theta^{2a (n_0 + n)} e^{- (n_0 t_0 + n \bar{t}) \theta^2} \\ \\ & \qquad \text{where} \quad \bar{t} = \sum _i y^{2} _{i} \end{aligned} \end{split}\]

となる。

c#

比を計算する。

\[\begin{split} \begin{aligned} p(\theta _a | y_1, \cdots, y_n) / p(\theta _b | y_1, \cdots, y_n) &= \frac{ \theta^{2a(n_0 + n)}_{a} e^{-(n_0 t_0 + n \bar{t}) \theta^{2}_{a}} }{ \theta^{2a(n_0 + n)}_{b} e^{-(n_0 t_0 + n \bar{t}) \theta^{2}_{b}} } \\ &= \left( \frac{\theta_{a}}{\theta_{b}} \right) ^{2a(n_0 + n)} \exp\left\{{-(n_0 t_0 + n \bar{t}) (\theta^{2}_{a}-\theta^{2}_{b}) } \right\} \end{aligned} \end{split}\]

これは \(n\bar{t} = \sum_{i} y^{2}_{i}\) の関数であるから、十分統計量は \(\sum_{i} y^{2}_{i}\) である。

d#

\(\theta\) の事後分布は \(\text{Galenshore}(a(n_0 + n) + 1/2, \sqrt{n_0 t_0 + n \bar{t}})\) であるから、

\[ E[\theta|y_1, \cdots, y_n] = \frac{ \Gamma \left( a(n_0 + n) + 1\right) }{ \Gamma \left( a(n_0 + n) + 1/2 \right) \sqrt{n_0 t_0 + n \bar{t}} } \]

となる。

e#

予測分布は以下のように計算できる。

\[\begin{split} \begin{aligned} p(\tilde{y}|y_1, \cdots, y_n) % 1個目の等号 &= \int d\theta ~ p(\tilde{y}| \theta) ~ p(\theta | y_1, \cdots, y_n) \\ % 2個目の等号 &= \int d\theta \frac{2}{\Gamma(a)} ~ \theta ^{2a} \tilde{y}^{2a-1} e^{-\theta^2 \tilde{y}^2} \\ & \qquad \times \frac{2}{\Gamma(a (n_0 + n) + 1/2)} (n_0 t_0 + n \bar{t})^{a(n_0+n) + 1/2} \theta^{2a (n_0 + n)} e^{- (n_0 t_0 + n \bar{t}) \theta^2} \\ % 3個目の等号 &= \int d\theta \frac{4}{\Gamma(a)\Gamma(a(n_0 + n) + 1/2)} (n_0 t_0 + n \bar{t} + \tilde{y}^2)^{a(n_0 + n + 1) + 1/2} \theta ^{2a(n_0 + n + 1)} e^{-(n_0 t_0 + n \bar{t} + \tilde{y}^2) \theta^2} \\ % 4個目の等号 &=\frac{2 \Gamma(a(n_0 + n + 1) + 1/2)}{\Gamma(a)\Gamma(a(n_0 + n) + 1/2)} \frac{ (n_0 t_0 + n \bar{t})^{a(n_0 + n) + 1/2} }{ (n_0 t_0 + n \bar{t} + \tilde{y}^2)^{a(n_0 + n + 1) + 1/2} } \tilde{y}^{2a-1} \\ & \qquad \times \int d \theta \underbrace{ \frac{2}{\Gamma(a(n_0 + n + 1) + 1/2)} (n_0 t_0 + n \bar{t} + \tilde{y}^2)^{a(n_0 + n + 1) + 1/2} \theta ^{2a(n_0 + n + 1)} e^{-(n_0 t_0 + n \bar{t} + \tilde{y}^2) \theta^2} }_{\text{Galenshore分布の密度関数だから積分したら1}} \\ % 5個目の等号 &= \frac{2}{ B(a, a(n_0 + n) + 1/2) } \frac{ (n_0 t_0 + n \bar{t})^{a(n_0 + n) + 1/2} }{ (n_0 t_0 + n \bar{t} + \tilde{y}^2)^{a(n_0 + n + 1) + 1/2} } \tilde{y}^{2a-1} \\ % 6個目の等号 &= \frac{ 2 \left( \frac{\tilde{y}}{{\sqrt{n_0 t_0 + n \bar{t}}}} \right) ^{2a -1} \left[ 1 + \left( \frac{\tilde{y}}{{\sqrt{n_0 t_0 + n \bar{t}}}}\right) ^ {2} \right] ^{-a -[a(n_0 + n) + 1/2]} }{ \sqrt{n_0 t_0 + n \bar{t}} B(a, a(n_0 + n) + 1/2) } \end{aligned} \end{split}\]

これはパラメータが

\[\begin{split} \begin{cases} \alpha = a \\ \beta = a(n_0 + n) + 1/2 \\ p = 2 \\ q = \sqrt{n_0 t_0 + n \bar{t}} \end{cases} \end{split}\]

の一般化第2種ベータ分布である。

参考: Galenshore分布とGamma分布と一般化Gamma分布#

\(\text{Galenshore}(a, \theta)\) の密度関数 (1) を \(y^2 = z\) と変数変換してみる。
変数変換した後の変数 \(z\) の密度関数 \(p_{Z}(z)\) は

\[\begin{split} \begin{aligned} p_{Z}(z) &= p(y(z)) \left|\frac{dy}{dz}\right| \\ &= \frac{2}{\Gamma(a)} \theta ^{2a} z^{a-1/2} e^{-\theta^2 z} \times \frac{1}{2} z^{-1/2} \\ &= \frac{(\theta^2) ^{a}}{\Gamma(a)} z^{a-1} e^{-\theta^2 z} \end{aligned} \end{split}\]

となる。これはパラメータ \(a=a, b=\theta^2\) のGamma分布 \(\text{Gamma} (a, \theta^2)\) である。

確率変数 \(X\) が一般化Gamma分布 \(\text{GG}(a, d, p)\)に従うとき、密度関数は

\[ p(x) = \frac{ (p/a^d)x^{d-1}e^{-(x/a)^p} }{ \Gamma(d/p) } \]

で表される。ここで、\(a, d, p > 0\) がパラメータである。 (1) と見比べると \(\text{Galenshore}(a, \theta)\) = \(\text{GG}(1/\theta, 2a, 2)\) であることがわかる。
